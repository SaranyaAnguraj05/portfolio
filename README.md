## Resume Download

The resume download button uses your Google Drive link:
- Clicking "Download Resume" will directly download the file
- The link is configured to work with the Google Drive ID provided

## Customization Tips

If you want to customize colors, edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #4facfe;
}
```
